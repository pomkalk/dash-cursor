# Documents Dashboard (React + TypeScript + Vite)

Локальный запуск проекта.

## Требования

- `Node.js` 20+ (рекомендуется LTS 20/22)
- `npm` 10+

Проверить версии:

```bash
node -v
npm -v
```

## Установка зависимостей

В корне проекта:

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

После запуска открой адрес из терминала (обычно `http://localhost:5173`).

## Сборка production-версии

```bash
npm run build
```

## Предпросмотр production-сборки

```bash
npm run preview
```

## Линтинг

```bash
npm run lint
```
