# StackedProgressBar

- **Path:** `src/components/StackedProgressBar/index.tsx`
- **Role:** отображение распределения категорий в одной горизонтальной полосе.

## Props
- `segments: { value: number; color: string; label: string }[]`.

## Behavior
- Считает общий `total`.
- Показывает процентные подписи и визуальные сегменты по ширине.
