# DataTable

- **Path:** `src/components/DataTable/index.tsx`
- **Role:** универсальная table-обертка с прокидываемыми head/body.

## Props
- `head: ReactNode` — содержимое `thead`.
- `body: ReactNode` — содержимое `tbody`.

## Behavior
- Добавляет border/scroll container и базовую типографику таблицы.
- Используется в `RecentFilesTable`.
