# KpiStat

- **Path:** `src/components/KpiStat/index.tsx`
- **Role:** компактная KPI-ячейка (число + подпись) для аналитических блоков.

## Props
- `label: string`.
- `value: number`.
- `tone?: 'neutral' | 'success' | 'warning'`.

## Behavior
- Унифицирует мини-статистику в `DocumentComplianceCard`.
