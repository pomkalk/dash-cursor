# FolderCard

- **Path:** `src/components/FolderCard/index.tsx`
- **Role:** карточка категории документов в верхнем гриде.

## Props
- `title: string`.
- `filesCount: number`.
- `totalSizeLabel: string`.
- `onOpen?: () => void`.

## Behavior
- Имеет hover-подъем и action-кнопку открытия.
- Используется внутри `FolderCardsSection`.
